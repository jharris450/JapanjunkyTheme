# Audio Sources

Sources for the forest scene audio. All files must be CC0, CC-BY, or PD.
Track each source URL + license + author here before adding to this dir.

## Required files

Drop these into `assets/sounds/`. The audio module (`japanjunky-audio.js`)
auto-loads them on first user gesture. Missing files fail silently.

| File                  | What                                                       | Length    | Notes      |
|-----------------------|------------------------------------------------------------|-----------|------------|
| `ambient_forest.ogg`  | Wind through cedars + cicada/bird + dripping water + creak | ~40s loop | -24 LUFS   |
| `accent_chime.ogg`    | Soft wood-block + wind bell                                | ~1.2s     | bright     |
| `accent_paper.ogg`    | Paper rustle (shide)                                       | ~0.6s     | crisp      |
| `accent_stone.ogg`    | Dry stone tap                                              | ~0.4s     | percussive |
| `accent_bell.ogg`     | Distant temple bell                                        | ~2.0s     | reverberant|
| `accent_step.ogg`     | Gravel/needle crunch                                       | ~0.3s     | gritty     |
| `accent_tsuno.ogg`    | Breathy whisper                                            | ~0.8s     | unsettling |
